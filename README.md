# Comprehensive-Stats-ML-DL-TimeSeries-Notes
This repository contains my handwritten in-depth notes and hand-picked resources on a wide range of core and advanced topics in data science and AI, including:

- **Statistical Inference**
- **Regression Analysis**
- **Time Series Forecasting**
- **Machine Learning** 
- **Deep Learning**
- **Graph Neural Networks**
- **Spatiotemporal Modeling**

## Key Resources

| Resource | What You will Learn |
|----------|--------------------|
| [Introduction to Statistical Learning (ISLR)](https://www.statlearning.com/) | Foundations of regression, classification, model evaluation, and regularization with intuitive explanations and R examples. |
| [The Elements of Statistical Learning (Hastie, Tibshirani, Friedman)](https://hastie.su.domains/ElemStatLearn/) | A deeper, more mathematical complement to ISLR. Great for understanding regularization, kernel methods, and ensemble learning. |
| [Statistical Inference (Casella & Berger)](https://books.google.ae/books?id=FAUVEAAAQBAJ&printsec=frontcover) | Formal treatment of probability, estimation, hypothesis testing, likelihood methods, and asymptotic theory. |
| [All of Statistics (Larry Wasserman)](https://link.springer.com/book/10.1007/978-0-387-21736-9) | A concise yet comprehensive introduction to probability, estimation, testing, and Bayesian ideas. |
| [Bayesian Data Analysis (Gelman et al.)](https://www.taylorfrancis.com/books/mono/10.1201/9780429258411/bayesian-data-analysis-andrew-gelman) | The gold standard for Bayesian modeling. Covers hierarchical models, MCMC, and applied examples in science and social data. |
| [Understanding Machine Learning (Shalev-Shwartz & Ben-David)](https://mml-book.github.io/) | A theoretical foundation of ML including VC dimension, PAC learning, supervised/unsupervised algorithms, and generalization theory. |
| [The Hundred-Page Machine Learning Book (Burkov)](https://themlbook.com/) | A concise and practical overview of machine learning algorithms, key concepts, and real-world applications. Great for fast revision. |
| [Dive into Deep Learning (D2L)](https://d2l.ai/) | Hands-on deep learning guide using PyTorch/MXNet. Covers MLPs, CNNs, RNNs, optimization techniques, and applied projects. |
| [Deep Learning (Goodfellow, Bengio, Courville)](https://www.deeplearningbook.org/) | The definitive theoretical reference for deep learning: representation learning, backpropagation, optimization, and deep generative models. |
| [Forecasting: Principles and Practice (FPP3)](https://otexts.com/fpp3/) | Practical and modern time series forecasting with R. Covers ARIMA, exponential smoothing, trend/seasonality, and GEV-based modeling. |
| [Graph Neural Networks (Z. Wu et al.)](https://gnn.seas.upenn.edu/) | Key concepts and architectures in GNNs including GCNs, GATs, spectral methods, and applications to node and graph-level tasks. |
| [Deep Learning on Graphs (Yao Ma et al.)](https://yaoma24.github.io/dlg_book/) | Modern deep learning techniques for graph and spatiotemporal data. Covers theory, architectures, and real-world use cases with code. |
| [Spatiotemporal Data Analysis (Cressie & Wikle)](https://books.google.ae/books?id=zNLhAwAAQBAJ) | Statistical models for spatial and temporal processes, including geostatistics, hierarchical modeling, and dynamic systems. |
